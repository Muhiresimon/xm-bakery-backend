# XM Bakery Backend

This is the backend system for XM Bakeries, built using Node.js and MySQL.

## Features

- Manage bakery products
- Add, view, delete products
- Secure database connection

## Technologies Used

- Node.js
- Express
- MySQL
- Postman for API testing
